# irssi

* Homepage: http://irssi.org/

Irssi is a terminal based IRC client for UNIX systems. It also supports
 SILC and ICB protocols via plugins.

 Features include:

  * Autologging
  * Formats and themes
  * Configurable keybindings
  * Paste detection
  * Perl scripting
  * Irssi-proxy
  * Transparent upgrading
  * Recode support
